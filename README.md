# Retail_Sales_Analysis_Python

**Project Overview**
This project focuses on analyzing retail sales data to identify meaningful business insights, customer purchasing patterns, and sales trends using Python.
The objective of this project was not only to perform Exploratory Data Analysis (EDA), but also to understand how raw business data can be transformed into actionable insights that support decision making.

This project helped me strengthen my skills in:
- Data Cleaning
- Data Analysis
- KPI Reporting
- Data Visualization
- Business Insight Generation

  **Business Problem**
Retail businesses generate large amounts of transaction data every day.  
The challenge is to analyze this data to answer important business questions such as:

- Which products generate the highest revenue?
- Which countries contribute most to sales?
- When do customers purchase the most?
- Which customers contribute the highest revenue?
- What trends can help improve business performance?

**Tools & Technologies Used**
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

**Dataset Information**
The dataset contains online retail transaction records with columns such as:
- Invoice Number
- Product Description
- Quantity
- Unit Price
- Customer ID
- Invoice Date
- Country

**Project Workflow**
1. Data Cleaning & Preprocessing
Performed multiple cleaning steps to improve data quality:
- Removed missing values
- Removed duplicate records
- Removed cancelled/negative transactions
- Removed operational transactions
- Converted data types
- Created new calculated columns

2. Feature Engineering
Created additional columns for better analysis:
- Revenue
- Month
- Year
- Hour

3. Exploratory Data Analysis (EDA)
Performed analysis on:
- Monthly Revenue Trends
- Top Selling Products
- Country wise Revenue
- Peak Purchasing Hours
- Customer Purchase Behaviour
- Repeat Customers
- Revenue Distribution

**KPI Summary**
The following KPIs were calculated during analysis:
- Total Revenue
- Total Orders
- Total Customers
- Average Order Value
- Top Revenue Generating Countries
- Best Selling Products

  **Key Business Insights**
- The United Kingdom contributed the highest revenue.
- Certain products generated significantly higher sales volume.
- Sales activity increased during specific months and hours.
- Repeat customers played an important role in total revenue generation.
- A small number of products contributed a large share of revenue.

**Business Recommendations**
Based on the analysis, the following recommendations were identified:
- Focus marketing efforts on high performing products.
- Improve customer retention strategies for repeat buyers.
- Expand strategies used in top performing regions.
- Monitor low-performing products to improve inventory decisions.

Conclusion

This project helped me understand how real world retail data can be analyzed to generate meaningful business insights.
Through this project, I gained hands on experience in cleaning raw datasets, performing exploratory analysis, identifying trends, and presenting insights using Python.
